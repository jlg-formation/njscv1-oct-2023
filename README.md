# Gestion Stock

Application servant à gérer des articles.

## Install

```
git clone https://github.com/jlg-formation/njscv1-oct-2023.git gestion-stock
cd gestion-stock

cd front
npm i
npm run build

cd ..

cd back
npm i
```

## Start the back end server in production

```
cd gestion-stock
cd back
npm run prod
```
